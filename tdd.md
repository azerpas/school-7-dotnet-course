## Avantages
- Accélérer le développement
- Améliorer le développement
- Réduire les A/R QC

## Difficultés
- Une discipline
- Difficile sur l'existant

## Setup
- Créer un projet xUnit.net
- Ajouter une référence du projet Core

## Conseils
3 étapes:
- Mise en place
- Action à tester
- Tester les résultats
